## Installation and usage
 
**Clone respository:**
 
 ```
 https://github.com/mehmedayt/react-exam-project.git
 ```
 
 * To run client app
   <br/>
 ```
    cd client 
    npm install
    npm run dev
 ```
 * To run server app
    <br/>
 ```
    cd server
    npm install
    node ./server.js
 ```
